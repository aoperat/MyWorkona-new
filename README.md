# MyWorkona

Workona 유사 탭 관리 Chrome 확장 프로그램

## 기능

- 워크스페이스 관리: 여러 워크스페이스를 생성하고 관리할 수 있습니다
- 탭 저장 및 복원: 현재 열린 탭을 워크스페이스에 저장하고 나중에 일괄 복원할 수 있습니다
- 리소스 관리: 자주 사용하는 사이트를 워크스페이스에 고정할 수 있습니다
- 노트 기능: 워크스페이스별 메모를 작성할 수 있습니다
- 검색 기능: 저장된 탭을 검색할 수 있습니다
- Google 계정 동기화: Chrome Sync Storage를 사용하여 데이터를 Google 계정으로 동기화합니다

## 설치 방법

1. 저장소를 클론하거나 다운로드합니다
2. 의존성을 설치합니다:
   ```bash
   npm install
   ```
3. 프로젝트를 빌드합니다:
   ```bash
   npm run build
   ```
4. Chrome 브라우저에서 확장 프로그램 관리 페이지(`chrome://extensions/`)를 엽니다
5. "개발자 모드"를 활성화합니다
6. "압축해제된 확장 프로그램을 로드합니다"를 클릭합니다
7. `dist` 폴더를 선택합니다

## 개발

개발 모드로 빌드하려면:

```bash
npm run dev
```

이 명령은 파일 변경을 감지하고 자동으로 빌드합니다.

## 아이콘

아이콘 파일은 `public/icons/` 디렉토리에 다음 크기로 준비해야 합니다:
- icon16.png (16x16)
- icon32.png (32x32)
- icon48.png (48x48)
- icon128.png (128x128)

현재는 placeholder 파일이 있으므로, 실제 아이콘 이미지로 교체해주세요.
